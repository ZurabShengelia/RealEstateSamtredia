🏠 RealEstateSamtredia 
A modern and functional Real Estate Management Dashboard with integrated contact features

✨ Key Features
Property Grid: Dynamic listings with smart unit logic ($ vs ₾, sq ft vs m²).

Fully Responsive: Optimized for Mobile, Tablet, and Desktop with a smooth Burger Menu.

Favorites System: Save your favorite properties to a dedicated section.

Email Integration: A functional contact form that sends messages directly to the admin's email via Node.js.

Smart Sidebar: Easy navigation between Dashboard, Analytics, Favorites, and Messages.

Dark Mode: Fully optimized UI for light and dark environments.

🛠️ Technology Stack
Frontend: HTML5, CSS3, JavaScript (ES6).

Backend: Node.js.

Icons: FontAwesome.

🚀 How to Use
1.Install Node.js (If not already installed).

2.Install npm on your terminal.

3.Write cd Backand at first

4.On terminal write: node server.js

5.Server is ready! Open the dashboard in your browser.
